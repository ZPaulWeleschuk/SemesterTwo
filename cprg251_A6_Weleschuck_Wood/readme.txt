Project Title: Movie Management System using JDBC(CPRG 251 Assignment 6)

Program Description: This program is a system to manage a database of movies.
This program reads the current list of movies in the database using the MariaDBDriver.
This program allows the user to add movies to the database and saves the new movie to the database at the end.
This program allows the user to search the database based for all movies that released in a user specified year.
This program allows the user to generate a list of random movies.
This program also allows the user to delete any row in the database

Date:May 17, 2020

Authors: Joel Wood and Zennon Weleschuk

How to run program: Run the appDriver class and follow the on screen prompts.
