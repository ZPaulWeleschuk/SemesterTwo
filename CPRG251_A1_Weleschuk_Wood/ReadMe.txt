Project Title: Movie Management System (CPRG 251 Assignment 1)

Program Description: This program is a system to manage a list of movies.
This program reads the current list of movies in the database from text file.
This program allows the user to add movies to the database and saves the new movie to the text file.
This program allows the user to search the database based for all movies that released in a user specified year.
This program also can generate list of random movies of any length. 

Date:May 17, 2020

Authors: Joel Wood and Zennon Weleschuk

How to run program: Run the Appdriver class and follow the on screen prompts.
